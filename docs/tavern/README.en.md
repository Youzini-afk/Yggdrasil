# SillyTavern successor project

> [English](./README.en.md) · [中文](./README.md)

The next-generation SillyTavern lives in its own repo: **YdlTavern** at <https://github.com/Youzini-afk/Yggdrasil-Tavern>.

It runs on top of Yggdrasil and consumes the platform through the public protocol. It doesn't live in this repo — platform and product stay separate.

- [`TAVERN_COMPAT.md`](TAVERN_COMPAT.en.md) — Yggdrasil's boundary and integration stance toward YdlTavern.

For upstream TavernHeadless research, see [`../../integrations/tavern-headless/`](../../integrations/tavern-headless/). For generic capability packages distilled from Tavern lessons, see [`../guides/CREATIVE_CAPABILITY_KIT.md`](../guides/CREATIVE_CAPABILITY_KIT.en.md).
